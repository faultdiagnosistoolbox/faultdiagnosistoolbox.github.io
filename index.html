Fault Diagnosis Toolbox

Fault Diagnosis Toolbox is a Matlab toolbox for analysis and design of fault diagnosis systems for dynamic systems, primarily described by differential-algebraic equations. Key features of the toolbox are extensive support for structural analysis of large-scale dynamic models, fault isolability analysis, sensor placement analysis, and code generation in C/C++ and Matlab.

For a quick introduction, see the use case where an industrial size example, an automotive engine, is analyzed, C-code for residual generators is generated, and the resulting diagnosis system is evaluated on test-cell measurements from our engine laboratory.
